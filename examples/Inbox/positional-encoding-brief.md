# Topic Brief: Positional Encoding in Transformers

Build a short set of notes on how transformers inject order information into a
permutation-invariant attention mechanism.

Cover:
- Why self-attention has no inherent notion of token order
- Sinusoidal positional encodings (Vaswani et al., 2017)
- A brief contrast with learned-absolute and relative/rotary schemes

Audience: someone who understands attention but not how position is handled.
